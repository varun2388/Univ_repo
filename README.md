# Univ_repo

## Machine Learning Model using Iris Dataset

This repository contains a machine learning model built using the Iris dataset. The Iris dataset is a classic dataset in the field of machine learning and statistics. It contains 150 samples of iris flowers, with four features (sepal length, sepal width, petal length, and petal width) and three classes (Iris-setosa, Iris-versicolor, and Iris-virginica).

The model is built using the `scikit-learn` library in Python. It includes the following steps:
1. Loading the Iris dataset
2. Preprocessing the data
3. Building a machine learning model
4. Training and evaluating the model

## Deployment using Streamlit

The machine learning model is deployed using Streamlit, an open-source app framework for Machine Learning and Data Science projects. Streamlit allows us to create interactive web applications with ease.

To run the Streamlit app, use the following command:
```
streamlit run app.py
```

The app provides an interface for users to input the features of an iris flower and get the predicted class (Iris-setosa, Iris-versicolor, or Iris-virginica) from the trained machine learning model.
